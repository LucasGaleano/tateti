# TaTeTi

Divertido juego para todos.

## requerimientos
- python3
- teminal de linux

## Como instalar
`git clone https://github.com/LucasGaleano/tateti`\
`cd tateti`\
`chmod +x tateti`\
`./tateti`

## Como jugar
Cuando sea tu turno utlizar los numeros para seleccion la posicion\
7|8|9\
4|5|6\
1|2|3
